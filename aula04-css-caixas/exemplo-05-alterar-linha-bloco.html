Alterar em linha ou em bloco e visibilidade de caixas

A propriedade display permite transformar um elemento de linha em um elemento de bloco ou vice-versa. Valores:

inline
transforma um elemento de bloco em um elemento de linha
block
elemento de linha se transforma em um elemento de bloco
inline-block
o elemento se comporta como elemento de linha, mas mantem características de um elemento de bloco (por exemplo, pode-se estabelecer largura e altura: width, height, etc..)
none
oculta o elemento da página. Útil para não mostrar inicialmente um elemento na página, mas após interação com usuário (por meio de controle via código javascript), mostrar o conteúdo

A propriedade visibility permite ocultar caixas, mas deixa um espaço onde o elemento estaria. Os valores podem ser hidden ou visible.
