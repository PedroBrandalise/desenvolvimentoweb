Layout

Conceitos chave para posicionamento de elementos

CSS trata cada elemento como estivesse em uma caixa própria. A caixa pode ser em nível de bloco ou uma caixa em linha.

Elementos containers: se um elemento de nível de bloco estiver dentro de outro elemento de nível de bloco, então a caixa externa é conhecida como elemento container ou elemento pai. Um elemento pai ou container é definido por meio de um elemento div.

CSS tem os seguintes esquemas de posicionamento, que são especificados pelo elemento position:

  1. Fluxo normal (position:static)
  2. Posicionamento relativo (position:relative)
  3. Posicionamento absoluto (position:absolute)
  4. Posicionamento fixo (position:fixed)
  5. Sobrepondo elementos (zindex)
  6. Elementos flutuantes (float:right, float:left...)
  7. Elementos lado a lado com elementos flutuantes
  8. Isolando elementos flutuantes (clear:left)
  9. Criando layouts multicolun com elementos flutuantes (float:)
  10. Exemplo com três colunas

Layouts: largura fixa e fluídos

Designs com layout de largura fixa não mudam de tamanho quando o usuário aumenta ou diminui o tamanho da tela do navegador. As dimensões tendem as ser dadas em pixels.

Designs com layouts fluídos (líquidos) estendem e compactam à medida que o usuário aumenta ou dominui o tamanho da janela do navegadores. As dimensões são dadas em porcentagens.
